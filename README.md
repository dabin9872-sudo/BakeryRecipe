# BakeryRecipe

<카페_레시피북>
여러 카페가 입점해 상품(케이크/쿠키/빵)을 픽업 예약으로 판매하고, 회원은 레시피를 공개/회원전용으로 공유하는 플랫폼

사용자 : 일반회원 / 카페사장 / 관리자

[메인]
<탭1>
1. 카테고리 바로가기
- 케이크 / 쿠키 / 빵

2. 인기 카페 TOP 5
- 카드 표시 카페명, 주소(서울시 ㅇㅇ동/ 경기도 구리시)

3. 일간/주간 인기 상품 TOP 5
- 카드 표시: 상품명, 가격, 카페명, 카테고리

4. 카페 전체보기

<탭2>
1. 인기 레시피 TOP 5
- 카드 표시 : 제목, 작성자, 좋아요/댓글 수
=> 전체 공개=> 상세로 바로 이동
      회원 공개 => 로그인 안 했으면 로그인 창

2. 최신 레시피 TOP 5
- 카드 표시 : 제목 + 작성일

3. 레시피 더보기/작성하기 버튼


[고객 페이지]
1. 카페 목록
2. 카페 상세
3. 상품 상세 + 픽업 예약 주문
4. 내 주문 목록/상세
5. 레시피 목록
6. 레시피 상세(회원 전용의 경우 로그인 창)
7. 레시피 작성/댓글 작성

[사장 페이지]
1. 상품 관리
2. 주문 관리(상태 변경)
3. 통계

[관리자 페이지] 
1. 대시 보드
- 신규 가입자, 신규 카페, 주문 수, 주문 취소율
- 최근 신고/문의 건수

2. 회원 관리
- 회원 목록/검색(이메일, 닉네임, 상태,,)
- 상세: 가입일, 역할, 활동
- 조치: 계정 활성/정지, 역할 변경(일반<-> 카페 사장), 비밀번호 초기화

3. 카페 관리
- 카페 등록 신청 리스트
- 승인/반려, 노출 상태(활성/비활성)
- 카페 정보 수정(주소/연락처/소개)
- 카페 연결 관리

4. 콘텐츠 관리(레시피/댓글)
- 레시피 목록/검색
- 비공개 처리/삭제 (신고 누적/정책 위반)
- 댓글 삭제/블라인드
